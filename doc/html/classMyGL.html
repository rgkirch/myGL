Detailed Description

This is the topmost class for the program. Creating a new instance of this class means creating a new instance of the program. The main program should be as simple as running MyGL* mygl = new MyGL(); The main.cpp main can then interact with the program in different ways through mygl. mygl->getCurrentContext(); mygl->switchCurrentContext(Context);

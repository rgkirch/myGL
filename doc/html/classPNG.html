Detailed Description

The PNG class is for reading and writing '.png' files. There are two static functions called readPNG() and writePNG() that can be used without instantiating an instance of the class. This class was made to solve the problem that writePNG(readPNG(fileName)); writePNG in the above statement doesn't know the width and height of the image. Now the width and height are stored in the object itself.
